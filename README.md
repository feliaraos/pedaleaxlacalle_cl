# Pedalea X la Calle — Archivo histórico

Rediseño del sitio de **Pedalea X la Calle** como un sitio **estático** (HTML/CSS/JS, sin
WordPress, sin base de datos), pensado como un **archivo histórico** del movimiento
ciudadano por la convivencia vial en Santiago de Chile (2011–2019).

## Estructura

```
index.html        Portada-manifiesto
historia.html     Línea de tiempo (2011–2019)
css/estilo.css    Sistema de diseño (risográfico-editorial)
CNAME             Dominio de GitHub Pages (rescate.pedaleaxlacalle.cl)
```

## Publicación

Sitio publicado con **GitHub Pages** desde la rama `main`, sobre el dominio
`rescate.pedaleaxlacalle.cl`.

## Filosofía

El contenido se organiza por **temas y procesos históricos**, no por orden cronológico de
blog. Criterio rector: *¿esta publicación ayuda a comprender la historia de Pedalea X la
Calle?* Ver el arco central en `historia.html` — la campaña por la Ley de Convivencia Vial
y la reducción de la velocidad máxima (#60MATA).
